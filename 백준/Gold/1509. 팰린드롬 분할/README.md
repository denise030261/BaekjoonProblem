### 문제 설명

<p>세준이는 어떤 문자열을 팰린드롬으로 분할하려고 한다. 예를 들어, ABACABA를 팰린드롬으로 분할하면, {A, B, A, C, A, B, A}, {A, BACAB, A}, {ABA, C, ABA}, {ABACABA}등이 있다.</p>

 <p>첫째 줄에 문자열이 주어진다. 이 문자열은 알파벳 대문자로만 이루어져 있고, 최대 길이는 2,500이다.</p>

### 문제 풀이
 <p>이 문제는 처음에는 그리디로 풀다가 적더라도 다른 영역을 팰린드롬 처리해야 분할의 개수가 더 적은 수를 구할 수 있기 때문에 dp로 풀어야 한다. 그래서 먼저 어느자리 부터 어느자리까지 팰린드롬인지를 구한다. 3글자 이상은 처음과 끝이 같고 그 사이가 팰린드롬인지 확인해야 팰린드롬이 맞는지 구할 수가 있다. 그렇게 구한 다음, for문을 통해서 0번째부터 i번째까지 팰린드롬인지 구하고 맞다면 i의 dp값이 1로 처리하고 만약 그렇지가 않다면 i-1번째부터 1번자리까지의 문자부터 i까지의 자리가 팰린드롬인지 파악한다. 맞다면 현재 자리의 dp의 값과 처음자리의 이전의 dp값에 1을 더해서 누가 더 적은지 파악해서 dp값을 갱신한다. 이전 자리의 1을 더한 이유는 이전의 팰린드롬의 분할 개수를 구해놓았기 때문이다.  </p>
