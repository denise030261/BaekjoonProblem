### 문제 설명

<p>N을 입력받은 뒤, 구구단 N단을 출력하는 프로그램을 작성하시오. 출력 형식에 맞춰서 출력하면 된다.</p>

### 입력 

 <p>첫째 줄에 N이 주어진다. N은 1보다 크거나 같고, 9보다 작거나 같다.</p>

### 출력 

 <p>출력형식과 같게 N*1부터 N*9까지 출력한다.</p>

### 문제 풀이
 <p>입력을 받은 숫자를 이용하여 구구단을 구하는 문제로 for문을 통해서 출력을 한다.</p>
